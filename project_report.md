# 🚀 專案報告：AI 智慧私人秘書 (AI Personal Secretary)

這是一個結合 **Google 生態系** 與 **Gemini AI** 的全方位個人助理應用程式。旨在透過自然語言處理技術，簡化使用者的日常生活管理，提供極致流暢的行動端體驗。

---

## 🎨 專案概覽 (Project Overview)

本專案是一個行動優先 (Mobile-First) 的 Web 應用程式，整合了行事曆、記帳、待辦清單與願望管理四大核心功能。其最大特色在於 **「對話即操作」**，使用者只需輸入（或語音輸入）一句簡單的話，AI 就能自動解析意圖並執行相應動作。

-   **開發目標**：解決跨平台切換 App 的繁瑣感，建立一個統一的個人資訊中控台。
-   **核心價值**：自動化、直覺化、美學化。

---

## 🛠 核心技術棧 (Technology Stack)

-   **Backend**: Python / Flask
-   **AI Engine**: Google Gemini Flash API (自然語言意圖解析)
-   **Database / Storage**: 
    *   **Google Calendar API**: 實時同步個人行程。
    *   **Google Sheets API**: 作為輕量化資料庫，儲存記帳、待辦與願望數據。
-   **Frontend**: Vanilla JavaScript (ES6+), HTML5, CSS3
-   **Voice Service**: Web Speech API (語音識別輸入)
-   **Design Aesthetic**: Tiffany Blue & Morandi 現代配色、自定義高級彈窗系統、全圖示化操作。

---

## ✨ 核心功能亮点 (Key Features)

### 1. 🤖 智慧意圖解析 (AI Intent Parsing)
整合 Gemini 1.5 Flash，能精準識別使用者的模糊輸入：
-   *「明天下午兩點在台北車站跟客戶開會」* → 自動轉換為 Google Calendar 行程。
-   *「今天晚餐吃了 500 元」* → 自動歸類為「食」並記錄至 Google Sheets，同時產出當月收支報告。

### 2. 🗓 雙向同步行程管理 (Smart Schedule)
-   **實時同步**：與 Google Calendar 深度整合。
-   **一鍵完工**：支援在畫面上直接標記行程完成（加註 ✅），並即時回寫至日曆。
-   **智能衝突檢查**：新增行程時，系統會自動檢查該時段是否有既有衝突。

### 3. 💰 專家級手動記帳 (Financial Tracker)
-   **內建計算機**：記帳欄位支援加減乘除運算。
-   **視覺化報告**：自動生成本月消費佔比與結餘比例，幫助使用者掌控財務狀況。

### 4. 📝 矩陣式待辦清單 (Priority Todo)
-   **優先級管理**：採用緊急/重要矩陣邏輯。
-   **自定義分類**：使用者可自由新增類別，並由 AI 建議對應的 Emoji 圖示。

### 5. 🌟 願望實踐清單 (Wishlist & Fulfill)
-   **預算追蹤**：自動計算所有願望的總預算。
-   **圓夢機制**：點擊「圓夢」即可記錄實際購入金額，讓「斷捨離」與「獲得感」數位化。

## 💎 UI/UX 設計理念

-   **極簡美學**：採用 Tiffany Blue (#00bfaf) 作為品牌色，搭配柔和的深灰色調與玻璃擬態 (Glassmorphism) 效果。
-   **行動端優化 (Bottom Sheet)**：針對手機單手操作，所有彈窗皆設計為從底部彈出的抽屜式結構。
-   **圖示化操作 (Iconography)**：全站採用統一風格的線條圖示 (SVG)，取代枯燥的文字按鈕，提升視覺直覺性。
-   **微互動體驗**：具備平滑的清單滑動消失動畫、按鈕縮放回饋，增強操作的「手感」。

---

## 📈 技術挑戰與解決方案

-   **挑戰**：AI 回傳格式不穩定。
    *   **解決方案**：設計了嚴格的 Prompt Engineering 與後端解析容錯機制，確保 JSON 輸出 100% 可用。
-   **挑戰**：Google Sheets API 的讀寫延遲。
    *   **解決方案**：前端實施 Optimistic UI (樂觀 UI) 更新，在請求發送時先行更新畫面，提升使用者體感的流暢度。
